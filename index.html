Display

TV mode uses larger fonts, focus rings, and D-pad navigation. Auto-detects based on device. The Computer option appears when running in a desktop browser, useful for previewing other modes from your laptop.

Plex Integration

Connect your Plex Media Server to mark items as in-library and launch playback. URL + token are pushed to your Cloudflare Worker (via "Save to Worker") and proxied from there — the seedbox no longer accepts cross-origin browser requests directly.

Plex Webhook Bridge

When you watch something on Plex, it tells CinéMath to mark it watched. Requires a Cloudflare Worker (free tier — see deployment instructions in the repo). Works with any Plex Pass account.
